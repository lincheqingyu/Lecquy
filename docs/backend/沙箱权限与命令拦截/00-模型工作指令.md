## 📋 项目背景

### 目标
设计和实现一套完整的**沙箱权限与命令拦截系统**，模仿 Claude Code 的成熟设计，融入 Lecquy 项目。

### 为什么这很重要？
- Claude Code 有 9400+ 行成熟的权限代码
- Lecquy 需要类似的防护机制来防止 AI 滥用
