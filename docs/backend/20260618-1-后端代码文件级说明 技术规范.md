## 可视化总览

```mermaid
flowchart TD
  FE[前端客户端 / 外部调用] -->|发起对话| CHAT[WebSocket 对话入口]
  CHAT -->|普通模式| SIMPLE[普通对话处理]
  CHAT -->|规划模式| PLAN[规划对话处理]

  SIMPLE --> AGENT[主智能体执行]
  PLAN --> MANAGER[规划智能体]
  MANAGER --> WORKER[执行智能体]

  AGENT -->|构建上下文和请求| PAYLOAD[模型请求组装]
  AGENT -->|会话状态| SESSION_RUNTIME[会话运行时]
  AGENT -->|读取记忆上下文| MEM_INJECT[记忆注入]
  AGENT -->|调用工具| TOOLS[工具层]
  AGENT -->|模型调用| MODEL[模型接口]

  PAYLOAD --> MODEL
  WORKER -->|工具执行| TOOLS

  MEM_INJECT -->|查询记忆| MEM_STORE[记忆存储]
  SESSION_RUNTIME -->|会话持久化| SESSION_DB[会话仓储]
  MEM_STORE --> MEM_DB[(SQLite / 兼容数据库)]
  SESSION_DB --> MEM_DB

  SESSION_RUNTIME -->|会话更新| PERM[权限中心]
  TOOLS --> PERM
  PERM --> AUDIT[审批审计]

  CHAT --> CONTROLLERS[HTTP 控制器]
  CONTROLLERS --> API[HTTP 接口/运维读写]
```
